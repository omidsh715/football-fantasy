# football fantasy

this a football fantasy website created with django


## Installation
for installation follow these steps 

first:
```bash
$ virtualenb venv
$ source venv/bin/activate
```
second:
```bash
$ pip3 install django
```
and finally
```bash
$ python3 manage.py makemigrations
$ python3 manage.py migrate
$ python3 manage.py runserver
```
